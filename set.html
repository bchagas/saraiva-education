SET - Saraiva Sistema de Ensino Técnico

O Saraiva Sistema de Ensino Técnico (SET) reúne os mais relevantes serviços, conteúdos e tecnologias para atender ao Programa Nacional de Acesso ao Ensino Técnico e EmpregoPronatec). Reconhecendo as particularidades do programa, o SET proporciona à instituiçãogerenciamento completo, da adesão ao SISTEC às estratégias para redução da evasão.

O SET oferece uma ótima relação custo/benefício e as melhores práticas para diferenciar os estudantes no mercado de trabalho.
